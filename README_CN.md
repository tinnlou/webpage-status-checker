# 网站链接状态检测工具 (Webpage Status Checker)

这是一个高效的网站链接状态检测工具，专为运营人员设计。它可以并发检测大量链接（支持 100 万+），并提供可视化的 Web 界面来管理和查看结果。

## 功能特点

*   **高性能并发**：使用 Python 异步技术，可同时检测数千个链接。
*   **智能去重**：上传文件时自动去除重复的链接，避免重复检测。
*   **断点续传**：支持暂停检测，并在调整配置（如并发数）后继续检测。
*   **失败重试**：自动重试失败的请求，减少因网络波动导致的误报。
*   **智能伪装**：随机切换 User-Agent，降低被目标网站识别和屏蔽的风险。
*   **实时监控**：提供 ETA（预计剩余时间）和实时错误日志，方便随时掌握检测状态。
*   **安全防封**：支持设置“每秒请求数 (RPS)”限制，避免因请求过快被服务器屏蔽。
*   **可视化界面**：简单易用的 Web 仪表盘，支持文件上传、参数配置和进度监控。
*   **结果导出**：检测完成后可一键下载 CSV 格式的详细报告（包含状态码和错误原因）。
*   **智能容错**：自动忽略 SSL 证书错误，减少误报；非 200 状态码会自动记录具体原因（如 Not Found, Internal Server Error）。

## 安装与启动

### 前置要求
*   电脑上需安装 [Python 3.10](https://www.python.org/downloads/) 或更高版本。

### 1. 安装依赖
在项目文件夹下打开终端（Terminal 或 CMD），运行以下命令：

```bash
# 创建虚拟环境 (可选，但推荐)
python3 -m venv venv
source venv/bin/activate  # Mac/Linux
# venv\Scripts\activate   # Windows

# 安装所需库
pip install -r requirements.txt
```

### 2. 启动工具
在终端中运行：

```bash
python main.py
```

看到 `Uvicorn running on http://0.0.0.0:8000` 字样说明启动成功。

## 使用说明

1.  **打开界面**：在浏览器中访问 [http://localhost:8000](http://localhost:8000)。
2.  **上传文件**：
    *   点击“选择文件”上传包含链接的 `.txt` 或 `.csv` 文件（每行一个链接）。
    *   如果不确定格式，可以点击“Download Template”下载模板文件。
3.  **配置参数**：
    *   **Concurrency (并发数)**：同时连接的数量，建议 50-100。
    *   **Req/Sec Limit (每秒请求限制)**：每秒发出的最大请求数，建议 100 左右。设置过高可能会被目标网站封禁 IP。
    *   **Timeout (超时时间)**：等待网页响应的最长时间（秒），默认 10 秒。
4.  **开始检测**：点击“Start Check”按钮。
5.  **查看结果**：
    *   进度条会实时显示检测进度。
    *   完成后，点击绿色“Download Results”按钮下载 CSV 报告。

## 常见问题

*   **Q: 为什么有些正常的链接显示 Error？**
    *   A: 请检查网络连接。本工具已默认忽略 SSL 证书错误。如果仍然报错，可能是目标网站有防火墙拦截了爬虫请求，尝试降低“并发数”和“每秒请求限制”。
*   **Q: 404 是什么意思？**
    *   A: 404 表示链接指向的页面不存在。在结果 CSV 中，Error 列会显示 "Not Found"。
